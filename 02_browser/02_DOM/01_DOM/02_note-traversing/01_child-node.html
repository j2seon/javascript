01 자식 노드 탐색

노트 탐색

        요소 노드(id,class tagName, CSS 선택) 후 취득 요소를 기점으로 DOM트리의 노드를 옮겨다니며 부모,형제 자식 노드등을 탐색 해야할 때가 많음
        DOM트리 상의 노드를 탐색할 수 있도록 Node, Element 인터페이스는 트리탐색 프로퍼티를 제공한다.

공백 텍스트 노드

        HTML 요소 사이의 스페이스, 행 줄바꿈(개행)등 공백 문자는 텍스트 노드를 생성한다.

자식 노드 탐색

    1
  1. Node.prototype.childNodes : 자식 노드(요소노드, 텍스트노드)를 탐색하여 NodeList에 담아 변환
    2. 2
  2. Node.prototype.firstChild : 첫번째 자식 노드(요소노드, 텍스트노트)반환
    3. 3
  3. Node.prototype.lastChild : 마지막 자식 노드(요소노드, 텍스트노드) 반환
    4. 4
  1. Element.prototype.children : 자식 노드 중 요소노드만 탐색하면 HTMLCollection에 담아 반환
  2. Element.prototype.fisrtElementChild : 첫번째 자식 요소 노드 반환
  3. Element.prototype.lastElementChild : 마지막 자식 요소 노드 반환
