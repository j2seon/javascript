01.노드 정보 취득

        Node.prototype.nodeType : 노드 객체의 종류 즉 노드타입을 나타내는 상수 반환
        Node.ELEMENT_NODE : 요소 노드 타입을 나타내는 상수 1 반환
        Node.TEXT_NODE : 텍스트 노드 타입을 나타내는 상수 3 반환
        Node.prototype.nodeName : 노드의 이름을 문자열로 반환한다
            요소노드 : 문자열로 태그 이름 반환
            텍스트 노드 : 문자열 "#text"반환
            문서 노드 : 문자열 #document 를 반환
