04. CSS선택자를 이용한 요소 노드 취득

        Document.prototype/Element.prototype.querySelector 메서드는 인수로 전달한 CSS 선택자를 만족시키는 하나의 요소들을 탐색하여 반환한다.
        메서드 이름이 Element가 복수형인것에 알 수 있읏 여러개의 요소 노드 객체를 갖는 DOM컬랙션 객체은 HTMLCollection 객체를 반환한다.

            Document.prototype/Element.prototype.querySelectorAll 메서드는 인수로 전달한 CSS 선택자를 만족시키는 모든 요소들을 탐색하여 반환한다.
            메서드 이름이 Element가 복수형인것에 알 수 있읏 여러개의 요소 노드 객체를 갖는 DOM컬랙션인 NodeList 객체를 반환한다.
